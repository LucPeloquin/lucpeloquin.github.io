Automatic Enhancement of Fashion Imagery

January - April 2024

A deep learning project focused on enhancing fashion product images using advanced computer vision techniques. Implemented a neural network model inspired by the FashionERN architecture to improve image quality and detail for e-commerce applications.

The system automatically enhances low-quality fashion product images, improving color accuracy, detail preservation, and overall visual appeal without manual intervention.

Technologies: Python, PyTorch, Computer Vision, Deep Learning, Fashion-MNIST dataset
