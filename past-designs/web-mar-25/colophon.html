Software

I built this site using vanilla HTML and CSS, embracing simplicity and performance. It's hosted through GitHub Pages and can be found at peloquin.dev. The source code is available in my GitHub repository.

For the design, I chose a minimalist approach that prioritizes readability and content. The monospaced IBM Plex Mono font gives the site a technical feel that aligns with my professional background.

My workflow involves drafting content in VS Code with Markdown before converting it to HTML. By keeping the site static and lightweight, I ensure fast loading times and straightforward maintenance.
